3.1 END-TO-END REQUEST FLOW

FRONTEND COMPONENT
  │ useQuery hook OR useEffect + direct service call
  ▼
REACT QUERY HOOK
  │ queryKey + queryFn + staleTime + retry
  │ Cache check → hit/fresh = skip network
  ▼
SERVICE LAYER
  │ async getXxx() { const { data } = await api.get('/path'); return data; }
  ▼
API CLIENT (Axios + api.ts)
  │ 1. REQUEST INTERCEPTOR: JWT Bearer token from localStorage
  │ 2. AbortController with timeout+1000ms
  │ 3. REQUEST DEDUPLICATION: Map<key, Promise>
  │ 4. VITE DEV PROXY: /api/* -> http://localhost:5000/api/*
  │ 5. RESPONSE INTERCEPTOR:
  │    CanceledError → mock fallback
  │    429 → retry ×3 (2s,4s,6s exponential)
  │    Network error/timeout → mock fallback + retry ×1 (3s delay)
  │    401 → POST /auth/refresh-token → retry original
  ▼
EXPRESS BACKEND
  │ 1. helmet → cors → morgan → rateLimiter
  │ 2. express.json(10mb) → urlencoded → cookieParser
  │ 3. Security headers (nosniff, DENY frame, XSS protection)
  │ 4. Route matching → authenticate middleware
  │ 5. Controller → Service → Prisma → PostgreSQL
  ▼
RESPONSE FLOWS BACK:
  res.json({ success: true, data: {...} })
  → Axios → Service returns response.data
  → React Query normalizes → Component receives { data, isLoading, isError }
